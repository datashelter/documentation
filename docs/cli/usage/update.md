# update

Updates the Snaper binary in place to the latest released version. Run it from the same location where `snaper` is installed; the command replaces the current executable.

```
Update Snaper to the latest version

Usage:
  snaper update [flags]

Examples:
./snaper update

Flags:
  -h, --help   help for update
```
